# 📱 Lawazia Pulse

🚀 Lawazia Pulse is a Flutter-based mobile application that integrates **OneSignal** to deliver real-time push notifications and display them dynamically in the app UI.

---

## ✨ Features

- 🔔 Real-time push notifications using OneSignal
- 📲 Dynamic notification display on homepage
- 💾 Local storage using SharedPreferences
- ☁️ Backend support with Supabase
- 🌙 Dark theme UI
- ⚡ Fast & responsive performance

---

## 🛠️ Tech Stack

- **Flutter (Dart)**
- **OneSignal SDK**
- **Supabase**
- **SharedPreferences**
